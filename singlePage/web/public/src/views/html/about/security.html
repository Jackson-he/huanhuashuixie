贷前审核
初次洽谈后，通过资料核实及实地考察的方式，对借款用户的借款用途、资信状况、偿还能力等进行分析，期间需通过下户查账（方法包括但不限于通过企业或个人财务报表、银行流水、上下游合同、科目余额表、二级明细账、水电缴费单、纳税报表及票据、租金缴纳情况、背景调查、重要谈话交流等）；身份证真伪查询；社保信息真伪查询；工作地址查询；个人及公司征信报告；公检法网站查询执行记录；抵押物价值及状态；工商系统查询等方式方法，后形成调研报告，进入审批环节。

评审报告
评审
为达到信贷专业化，龙贷特设立评审委员会，其成员七人，包括业务部经理、风险部经理、法务部经理、副总经理、总经理、财务部经理、首席风险官（评审主任）组成。会议根据项目量，由法务部提前进行通知并组织会议，项目经理就调研报告进行评述，后由评审委进行评议并签署意见，最终由评审主任确认结果，结果分为待定、通过、不通过三种形式，期间待定为存疑问题，需由项目经理补充调查；不通过则由法务部通知项目经理，此时项目终止；通过则进入放款流程。

    有庞大且大量的合作机构，其中包括资产管理公司、银行、担保机构等，可在风险预警同时，通过合作机构进行合理转化或债权转让，有效缓解或化解风险。
  • 借款项目多以房产为担保措施
    借款项目多以房产为担保措施，且均无贷款敞口，故违约率较低，处理效率较高。
“投资互助计划”是龙贷为保护平台全体理财人的共同权益而建立的信用风险共担机制。龙贷的所有理财人经过龙贷身份认证后，在龙贷的投资行为均适用于“投资互助计划”，理财人无需为此支付任何费用。“投资互助计划”为理财人提供了有效的风险共担机制，分散了理财人投资行为所带来的信用风险，营造了一个公平公开健康的投资环境，保护了理财人的权益。
*投资有风险，理财需谨慎。
龙贷不承诺投资项目本息100%无损失
风险准备金
是指在第三方平台在龙贷发布借款前，需预先按借款金额的一定比例放入其设立在第三方支付公司的“风险准备金账户”中。当理财人投资的借款一旦出现逾期时，龙贷将根据“风险准备金账户使用规则”通过“风险准备金”及时（最迟不超过30天）向理财人垫付此笔借款的剩余出借本金或本息（具体情况视投资标的类型的具体垫付规则为准）。
合作机构
是指除风险准备金方式外，对借款人进行管理、推荐、审核或者在龙贷进行债权资产转让的合作机构，根据各方签署的法律文件，对其转让、推荐和/或审核并由投资人购买、投资的债权资产（包括但不限于机构担保标）的偿付，通过提供包括但不限于担保、补足承诺、有限垫付、回购承诺的形式，保护投资人利益的实现。
信息完整披露
龙贷自有专业的风控团队对项目进行全方位尽职调查，确保信息的真实性和合法性。并且将项目信息、借款企业情况公开透明，投资人可根据项目信息进行风险判断，进行自主投资。
公司相关信息同样公开透明供投资人了解、判断。具体可见：关于龙贷
资金全程托管
风险准备金和交易资金由第三方全程托管，任何资金流动均不经过龙贷平台。
合乎法律法规
所有协议均经过专业律师起草并审核龙贷所有主营业务合同，制定充分保护投资人利益的条款。
自主开发的专业平台
龙贷网站访问全程采用VeriSign 256位SSL强制加密证书进行数据加密传输，有效保护用户个人信息、账户信息以及交易记录等机密信息在网络传输过程中不被查看、修改或窃取。
领先的数据异地灾备
同城多机房的实时备份，以及异地机房的实时备份，确保数据在任何时间的保密性、完整性及可用性。
完善的运维监测体系
在用户的账户发生异常访问或登录时，及时锁定账户及账户资金，并联系用户核实情况，全面保护用户信息及资金。
全面的敏感权限管理
我们严格遵守国家相关的法律法规，对用户的隐私信息进行保护。未经您的同意，龙贷不会向任何第三方公司、组织和个人披露您的个人信息、账户信息以及交易信息（法律法规另有规定的除外）。
同时，龙贷公司内部也设有严格、完善的权限管理体系，以保证每一位内部员工都只能查看自己职责和权限之内的数据和信息。
