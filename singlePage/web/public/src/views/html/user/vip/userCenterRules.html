龙贷会员是什么
为感谢各位用户对龙贷平台的支持，龙贷将为会员提供三元提现券、生日红包、升级礼物等多重权益。会员等级由成长值决定，会员等级越高，享受到的会员权益越大，专享特权越多。
会员等级介绍
会员等级 等级名称 成长值
V1 默默无闻 ≥0
V2 初入江湖 ≥100
V3 小有名气 ≥1000
V4 声名鹊起 ≥10000
V5 非同凡响 ≥100000
注:
  • 拥有会员等级对应的成长值即可升级；
  • 连续30天小于会员等级对应的成长值，自动降级；
  • 在法律法规许可的范围内，龙贷有权对会员规则进行解释。
举例:
小刚7月20日花10万投资一笔散标，7月21日获得V3等级，7月31日转出之前投资的10万债权，8月31日降为V1等级
成长值介绍
会员成长值 ≈ 待收本金/100
  • 龙贷用户每投资一笔项目（不限项目类型），可累计一次成长值，成长值在所投项目计息开始后加入账户；
  • 待收本金每增加100元，即可获得1点成长值；每减少100元，即减少1点成长值；
  • 成长值随待收本金增减后第二天生效；
  • 成长值按照待收本金金额除以100后的商取整数部分获得，小数点部分忽略不计；
  • 待收本金不包括投资时所用的红包、返利部分。
注:
龙贷平台未来将调整和优化龙贷会员对应的成长值加成体系，增加更多获得成长值的方法，敬请期待。
举例:
小明9月1日花1500元投资一笔散标，用了50元返利，9月2日可获得成长值14点
等级与特权
等级与特权 三元提现券 生日红包 升级加息券
V1 1张 10元 —
V2 1张/年 50元 +0.1%券1张/年
V3 2张/年 100元 +0.2%券1张/年
V4 3张/年 200元 +0.3%券1张/年
V5 4张/年 500元 +0.5%券1张/年
  • 每升一级，即可获得该等级对应的所有特权；
  • 如果直接升多级，则升级后获得当前等级及其之前多个等级下对应的所有特权（生日红包除外）；
  • 每个等级对应特权，从上次获得之日算起一年内只能获得一次；
  • 提现券和升级加息券可按需领取，一次领取一张，领取之后开始计算有效期。
举例:
小兵11月11日从V1升级到V3等级，之前没有领取过任何特权，故此时可领取特权有：4张提现券、1个生日红包（限生日当天领取）、2张加息券，当在11月12日领取1张0.1%加息券后，下一年11月12日前无论获得什么等级，都不可再领取0.1%加息券了
