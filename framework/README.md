#### AngularJS
    * angularJS的双向数据绑定技术真是棒极了,将精力放在对数据的操作上而不必关心它究竟是怎样做到的,因为angularJS能够快速入门.
    * 入门之后的angularJS想要用的得心应手并不容易,就像已经封装好的页面元素table,要不了多久我们就会觉得用受掌控的div绘制table,比去修改本身的table元素更令我们方便.
    * 但是绝不否认的是,angularJS对数据的处理绝对一流
